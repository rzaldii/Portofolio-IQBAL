Projects

BoostGro Mart

BoostGro Mart is a console based information system for agrocurtural product sales. The system is designed to make it easier for people to buy agricultural products at lower prices, as farmers can sell their products directly. The system was developed using Basic Algorithms and Python programming language with Visual Studio Code software.

Cands Laundry

Cands Laundry is a console based laundry service management information system at cands laundry. The purpose of creating this system is to make it easier for employees to record customer transaction data, because initially they recorded it manually in a book. The system was developed using Basic Algorithms, Basic SQL programming and Python programming language with Visual Studio Code software, also using Postgre SQL for the database.

MovieLand

MovieLand is movie ticket booking application as a solution of queue digitalization and transaction efficiency. The purpose of this system is to make transactions more efficient, and customers do not need to queue to purchase tickets. The application was developed using OOP Concept, Basic SQL programming and C# programming language with Microsoft Visual Studio software, also using Postgre SQL for the database and Canva for UI Design.
